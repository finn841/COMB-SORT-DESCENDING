**Comb Sort Algorithm in C++ (Descending Order)**

**Introduction**
This project demonstrates a simple implementation of the Comb Sort algorithm using C++. The program sorts numbers entered by the user in descending order. Comb Sort is an improvement over Bubble Sort. Instead of comparing only neighboring elements, it compares elements that are far apart using a “gap,” which moves small values toward their correct positions faster and makes sorting more efficient.

**How the Program Works**
1.The user enters the number of elements.
2.The user inputs the elements to be sorted.
3.The program applies the Comb Sort algorithm:
  -Elements are compared using a gap value.
  -The gap reduces gradually until it becomes 1.
  -At gap = 1, the array is fully sorted in descending order.
4.The program displays the sorted array.

**Features**
 -Sorts numbers in descending order.
 -Efficiently handles small to medium-sized arrays.
 -Easy to understand and modify for learning purposes.
